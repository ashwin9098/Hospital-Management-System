# Hospital Management System

# Overview
Hospital Management System ek web-based application hai jo hospital ke daily operations manage karta hai jaise patient records, doctor management, appointments, billing, etc.

#  Features
- Patient Registration & Records
- Doctor Management
- Appointment Scheduling
- Billing System
- Admin Dashboard

#  Tech Stack
- Frontend: Angular / React
- Backend: Node.js / Java / PHP
- Database: MySQL / MongoDB
